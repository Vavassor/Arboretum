Maybe Types
===========

.. c:type:: MaybeDouble

    An optional type containing a double precision floating-point number
    (``double``) and whether that number is valid.

.. c:type:: MaybeFloat

    An optional type containing a single precision floating-point number
    (``float``) and whether that number is valid.

.. c:type:: MaybeInt

    An optional type containing an integer (``int``) and whether that integer is
    valid.

.. c:type:: MaybePointer

    An optional type containing a pointer (``void*``) and whether that pointer
    is valid.

.. c:type:: MaybeUint64

    An optional type containing a 64-bit unsigned integer (``uint64_t``) and
    whether that integer is valid.
